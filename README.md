## Como Executar

1. Clone o repositório:
```sh
   git clone https://github.com/GuiHenry17/FUTGRIFE-DATABASE.git
```
2. Abra localmente
3. Instale as dependências do servidor e cliente:
 ```sh
   npm i
````
4. Inicie o servidor:
 ```sh
   npm start
````
5. Inicie o cliente:
 ```sh
   npm run dev
````
